# Termux cli
1. Attack las session
`$tmux a`
2. List all sessions
`$tmux ls`
3. Attach specific session`
`$tmux a -t <session-name>

# Termux shortcuts

 ## Sessions
 1. new session
`<C b> :new<enter>`
2. Detach 
`<C b> d`


# Windows resize
- Resize vertically
`<C b> + "`

- Resize horizontally
`<C b> + %`
